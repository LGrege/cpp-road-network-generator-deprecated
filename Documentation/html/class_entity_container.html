Detailed Description

Singleton Container class used by all NonTerminalSymbol classes.

Holds all segments, all intersection and other needed informations. Class is a singleton and is used allover the project as a shared information container (if necessary).

Note
Also holds functions for intersection checking
Some variables will be read from input file (capitalized variables)
